# Friday Initiatives Website

This repository contains the source code for the Friday Initiatives website, a data governance and value consulting firm.

## Overview

The website is a static HTML/CSS/JS site designed to showcase services, case studies, and company information.

### Structure

- **`index.html`**: Main landing page.
- **`pages/`**: Additional pages (About, Contact, News, Services, Privacy).
- **`assets/`**: Static assets including CSS stylesheets and JavaScript files.
- **`DEPLOYMENT_GUIDE.md`**: Instructions for deploying the website.
- **`QUICK_START.md`**: Guide for getting started with development.
